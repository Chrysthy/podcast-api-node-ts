<h1 align="center"> Podcast Manager 🎧 </h1> 

<p align="center">A video podcast manager in a Netflix-style. An open-source project built with NodeJS + TypeScript + HTTP Module that provides an API to organize video podcasts. Episodes can be listed by categories and filtered by podcast name, including details such as cover image, link, and title. 🎬</p>
